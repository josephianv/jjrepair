Crear tu base de datos

Firebase es gratis y guarda todos tus datos de forma segura en la nube.

Sigue estos pasos (5 minutos):

1️⃣ Abre console.firebase.google.com en otra pestaña

2️⃣ Haz clic en "Crear un proyecto" → ponle el nombre de tu tienda

3️⃣ Desactiva Google Analytics (no es necesario) → "Crear proyecto"

4️⃣ En el menú izquierdo → Build → Firestore Database → "Create database" → elige "Start in production mode" → selecciona cualquier ubicación → "Done"

5️⃣ En el menú izquierdo → Authentication → "Get started" → habilita "Email/Password"

6️⃣ En el menú izquierdo → Authentication → Users → "Add user" → crea tu cuenta con tu correo y contraseña

7️⃣ Arriba a la izquierda → ícono de ⚙️ (engranaje) → "Project settings" → baja hasta "Your apps" → haz clic en </> (Web) → ponle cualquier nombre → "Register app"

8️⃣ Copia el objeto firebaseConfig que aparece → pégalo en el siguiente paso

¡Ya casi listo!

Conectado a Firebase. Dos pasos finales y ya.

Paso final en Firebase Console:

1️⃣ Ve a Firestore Database → Rules y reemplaza todo el contenido con esto:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    function rol() {
      return get(/databases/$(database)/documents/Usuarios/$(request.auth.uid)).data.rol;
    }
    function esAdmin() { return request.auth != null && rol() == 'admin'; }
    function auth() { return request.auth != null; }
    match /Usuarios/{uid} {
      allow read: if request.auth != null && request.auth.uid == uid;
      allow write: if false;
    }
    match /{col}/{doc} {
      allow read: if auth();
      allow create, update: if auth();
      allow delete: if esAdmin();
    }
  }
}

2️⃣ Haz clic en "Publicar"

3️⃣ Ve a Firestore → Data → crea una colección llamada Usuarios (con U mayúscula) → dentro crea un documento con ID = el UID de tu cuenta (lo ves en Authentication → Users) → agrega un campo rol (string) = admin
